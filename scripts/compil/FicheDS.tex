\textbf{Commentaires généraux}
\begin{itemize}
\item C'était un DS largement abordable. Globalement, le résultat est assez décevant.
\item Seule une grosse moitié et un gros tiers d'entre vous sont capables de donner la résolution d'un codeur et de tracer un signal PWM. 
\item La méthode pour déterminer une fermeture géométrique n'est pas acquise pour tout le monde. 
\item Certains d'entre vous ne savent pas distinguer une liaison pivot glissant par rapport à une glissière, une liaison sphère-cylindre par rapport à une liaison ponctuelle.
\item Des confusions entre résultante et moment dans l'écriture des torseurs. 
\item Le paramétrage donné pour les rotules n'est quasiment jamais utilisé. 
\item Beaucoup trop d'erreurs sur les questions en asservissements. Beaucoup de questions étaient cependant accessibles. 
\item Les rapports de transmissions sont globalement à revoir : erreurs sur les signes des rapports, erreurs dans l'hélicoïdale. 
\end{itemize}
